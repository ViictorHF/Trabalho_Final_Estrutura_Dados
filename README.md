# EXPLICAÇÃO DOS PROGRAMAS -

O trabalho A consta de um código que executa, através de uma lista não ordenada, um menu 
que permite ao usuário cadastrar alguns dados, como nome de uma música, seu artista, duração, 
genero e seu id sendo esse ultimo dado usado como referencia para a busca, edição e 
remover da posição no qual foi inserido que são as outras opções do menu. O programa também usa 
da busca por índice em for e bubble-sort para executar algumas das opções do menu.

O trabalho B consta simula um call center para emergência com 
um menu que permite o usuário registra uma chamada, o local de ocorrência,
o tipo de emergência e horário, permite verificar a ultima chamada recebide,
lista todas as chamadas para atender e por fim atende-las.  

O trabalho C consta da simulação de um menu com possibilidade de
inserção de clientes de banco em duas filas, uma normal e outra
preferencial, podendo ver as filas, buscar um cliente especifica
remover um cliente do fim da fila e cancelar um cliente do meio
da fila.
